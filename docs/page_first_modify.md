1) “사용 흐름” 섹션 (교체)

> 목표: 두토의 핵심 강점(비전 + 프로젝트 기반 실행 구조화)을
> 고객이 이해하기 쉽게 4-step으로 전달한다.
> 톤: 쉬운 말, 부담 낮추기(비전은 선택)

사용 흐름 카피(최종)

1) 비전으로 인생의 방향을 잡아요 (선택)
“내가 어떤 사람이 되고 싶은지” 적으면, 목표가 흔들리지 않아요.  
*(비전이 없어도 바로 시작할 수 있어요.)*

2) 목표를 정해요
3개월 동안 이루고 싶은 목표를 한 줄로 정해요.

3) 프로젝트로 ‘실행 구조’를 만들어요 (핵심)
목표를 4주 프로젝트로 쪼개서, 이번 달에 끝낼 일을 선명하게 만들어요.

4) 오늘의 퀘스트를 완료해요
프로젝트를 오늘 할 수 있는 퀘스트(습관/과업)로 바꾸고,  
완료할수록 캐릭터가 성장하고 기록이 쌓여요.

(예시 1줄)
- 예: “5kg 감량(목표)” → “운동 루틴 만들기(4주 프로젝트)” → “주 3회 운동(퀘스트)”


2) “앱 미리보기” 섹션 (성장 시스템 미리보기)

> 기존의 “스크린샷 추후 업데이트 / 실제 화면 곧 공개” 등
> 미완성/예고성 문구는 모두 삭제한다.
> 대신 ‘캐릭터 4단계 진화’ 이미지를 메인 비주얼로 활용한다.

### 섹션 타이틀/카피 교체

- 타이틀: **앱 미리보기** → **성장 시스템 미리보기**

- 설명(2~3줄):
  - **퀘스트를 완료할수록 내 캐릭터가 성장해요.**
  - **알 → 새싹 → 개화 → 만개, 4단계 진화로 ‘진행감’을 눈에 보이게 만들어요.**

### 이미지 사용
- 캐릭터 성장 4단계 이미지(루미 단계 이미지) 1장 삽입  
  - 가능한 크게, 섹션의 메인 비주얼로 사용

### (선택) 보조 불릿 3개
- 퀘스트 완료 → 성장 포인트 획득
- 일정 기준 달성 → 다음 단계 해금
- 성장 기록은 리포트/피드백으로 이어져요 *(로드맵 적용 가능)*
